Game modes are different configurations that alter the number and types of roles in the game. They can also change certain other settings, such as the likelihood of a shaman receiving a specific totem.


The following is a list of the game modes currently available, and a description of the main differences compared to the usual rules (henceforth referred to as the default mode).

Main modes

These modes may be picked at random by the bot if there is no consensus in favor of any specific mode.

Name Description
Default The standard mode, used if no other mode fails to gain a majority of votes. See how to play for a description.
Foolish Contains the fool - be careful not to lynch them!
Mad Has the mad scientist, and many things that may kill you.
Classic The classic roles, from before the changes that evolved the game into its present form.
Lycan Contains lycans, who will turn into wolves. Hunt them down before the wolves overpower the village.
Aleatoire Contains a medley of roles on both sides, and modified totem chances for the shaman.
Alpha Contains the alpha wolf, who can turn others into wolves. Be careful who you trust!

Other modes

These modes cannot be selected randomly - a majority of users must vote for them.

Name Description
Rapidfire Contains many roles that lead to multiple chain deaths.)
Drunkfire Most players get a gun - quickly shoot all the wolves!
Noreveal Roles are not revealed when players die.
Amnesia Everyone is assigned a random role on night 3.
